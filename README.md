# Billory

Billory ist ein Tool zur digitalen Rechnungserstellung, Bearbeitung und Archivierung. Zusätzlich berechnet es Einnahmen und Steuern pro Quartal, damit Selbständige jederzeit einen klaren finanziellen Überblick haben.

---

## Zielgruppe & Zweck
- Selbständige und Solo-Unternehmer
- Nutzer, die Rechnungen digital erstellen und zentral verwalten wollen
- Personen, die ohne Buchhaltungssoftware Einnahmen- und Steuerwerte überblicken wollen

---

## Architektur / Technologien
| Bereich   | Technologie |
|----------|-------------|
| Backend  | Spring Boot |
| Frontend | React       |
| Datenbank | H2 (embedded) |


---

## Installation & Start

**Voraussetzung**
- Java **21** installiert (JDK oder JRE)

**Start (geplant)**
1. Java 21 installieren
2. Ausgelieferte `billory.exe` starten (Spring Boot + H2-DB laufen lokal)
3. Login durchführen
4. Browser öffnet Web-UI (z. B. http://localhost:8080)

> Anwendung ist aktuell **nicht funktionsfähig**. Entwicklung aktiv.

## Features (geplant / im Aufbau)
